# Proof of Alpha

Proof of Alpha is an experiment to track crypto alpha calls publicly.

The goal is to create transparency in crypto research.

Anyone can claim to have alpha, but very few people prove it.

This project logs early research and evaluates results over time.

## Idea

Users record early calls about crypto projects.

Each call has a timestamp.

Later we check if the thesis was correct.

## Vision

Create a reputation system for crypto researchers.
